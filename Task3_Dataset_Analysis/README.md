# Task 3 – Public Dataset Analysis

## Objective

The objective of this task is to load and analyze a public image dataset using PyTorch. The analysis includes examining the dataset structure, image properties, dataset distribution, and visualizing sample images with their corresponding labels.

## Dataset

- **Dataset Name:** Oxford-102 Flowers
- **Framework:** PyTorch (torchvision)
- **Total Images:** 8,189
- **Training Images:** 1,020
- **Validation Images:** 1,020
- **Testing Images:** 6,149
- **Number of Classes:** 102
- **Image Resolution:** 224 × 224 pixels

## Analysis Performed

- Downloaded and loaded the Oxford-102 Flowers dataset.
- Explored the dataset structure and image distribution.
- Examined image dimensions and color channels.
- Displayed sample images with their corresponding labels.
- Summarized the dataset characteristics and recorded observations.

## Technologies Used

- Python
- PyTorch
- Torchvision
- Matplotlib
- NumPy
